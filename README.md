# 🌤 Live Weather Forecast App

Real-time weather app using Open-Meteo API — **no API key required**.

## 🚀 Live Demo
[Dakshjain2006.github.io/weather-app](https://Dakshjain2006.github.io/weather-app)

## ✨ Features
- 🔍 Search any city worldwide
- 🌡 Current temperature, humidity, wind speed, visibility, feels-like
- 📅 5-day forecast
- 🎨 Dynamic theme switching based on weather (sunny / rainy / stormy / night / snowy)
- ⚡ Zero API key — uses free Open-Meteo API

## 🛠 Tech Stack
| Layer   | Technology                        |
|---------|-----------------------------------|
| Markup  | HTML5                             |
| Styling | CSS3, CSS Transitions             |
| Logic   | Vanilla JavaScript, Fetch API     |
| API     | Open-Meteo (weather + geocoding)  |

## 📁 Project Structure
```
weather-app/
├── index.html    ← Markup & layout
├── style.css     ← Styles + weather themes
├── script.js     ← API calls & DOM updates
└── README.md
```

## 🌐 APIs Used
- **Geocoding**: `https://geocoding-api.open-meteo.com/v1/search`
- **Weather**:   `https://api.open-meteo.com/v1/forecast`
